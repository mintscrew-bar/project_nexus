Nexus is a League of Legends community and match-management platform built for organized in-house games, tournaments, and player analytics.

Users can link their Riot accounts to verify identity, join or create rooms, participate in team selection flows such as auction or snake draft, select roles, and review completed matches. The platform also provides analytical views that compare ranked and custom games, show champion statistics, summarize player trends, and surface common builds and rune patterns.

We use Riot API data to support gameplay-related features only. Riot data powers account verification, match history display, match detail pages, timeline-based build analysis, champion statistics, and player comparison tools. The goal is to help players understand performance differences between ranked and custom games, improve match organization, and review game outcomes in one place.

Nexus does not use Riot data for unrelated purposes. Riot data is only used to operate the service features described above.

We use Riot API data for:
Summoner and account verification
Match history retrieval
Match detail display
Match timeline analysis
Champion and role statistics
Ranked vs custom game comparison
Build and rune trend analysis
Post-match review and analytics

We store only the data needed to provide the service:
Linked Riot account identifiers
Match metadata and normalized match participants
Aggregated champion statistics
Timeline-derived summaries for build analysis
Player-facing analysis results such as champion trends and build lists

We do not use Riot data for advertising, resale, or unrelated profiling.

We retain normalized match records and aggregated analytics so users can review historical performance over time. Raw Riot match cache data is kept only as long as needed for operational processing and can be cleaned up after normalization. If data deletion is needed, we can remove linked account and user-owned analysis data from the service.
