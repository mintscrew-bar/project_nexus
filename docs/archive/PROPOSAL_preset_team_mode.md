# 제안서: 사전 구성 팀 모드 (Preset Team Mode)

> 작성일: 2026-03-09
> 상태: 제안 (미착수)

---

## 배경

스크림, 대회, 사설 내전 등에서 팀이 이미 사전에 구성되어 있는 경우,
경매/드래프트 과정이 불필요하다. 이런 시나리오를 위한 새로운 팀 모드가 필요하다.

## 핵심 콘셉트

- **TeamMode: `PRESET`** 추가 (기존: AUCTION, SNAKE_DRAFT)
- 경매/드래프트 단계를 건너뛰고, **참가자 본인이 직접 팀을 선택**
- 롤 클라이언트의 사용자 설정 게임처럼 **팀 1 / 팀 2 로비**가 열리고, 각자 원하는 팀으로 이동
- 팀당 5명이 모두 차면 게임 시작 가능
- 역할 선택(Role Selection)은 **반드시 거침** — 전적 기록을 위해 필수

## 동작 흐름

```
방 생성 (teamMode: PRESET)
  → 로비 입장
  → 팀 선택 UI: [팀 1 (0/5)] [팀 2 (0/5)] [미배정]
  → 참가자가 본인이 원하는 팀 클릭하여 이동
  → 호스트가 "시작" → 양 팀 5명 충족 확인
  → 역할 선택 (Role Selection) 단계
  → 매치 진행
```

## UI 설계 방향

### 로비 — 팀 선택 화면

```
┌─────────────────────────────────────────────────┐
│                 [방 제목]                         │
├──────────┬───────────────┬──────────────────────┤
│  팀 1     │   미배정       │   팀 2               │
│  (3/5)   │              │   (2/5)              │
│          │              │                      │
│ ● 유저A   │  ● 유저F      │  ● 유저D              │
│ ● 유저B   │  ● 유저G      │  ● 유저E              │
│ ● 유저C   │              │                      │
│ [빈 슬롯]  │              │  [빈 슬롯]             │
│ [빈 슬롯]  │              │  [빈 슬롯]             │
│          │              │                      │
│  [← 이동] │              │  [이동 →]             │
└──────────┴───────────────┴──────────────────────┘
│              [게임 시작] (양 팀 5명 시)            │
└─────────────────────────────────────────────────┘
```

- 참가자는 **본인만** 이동 가능 (호스트도 타인 강제 이동 불가 — 공정성)
- 팀 클릭 또는 이동 버튼으로 자유롭게 팀 간 이동
- 5명 초과 시 해당 팀 이동 버튼 비활성화
- 관전자는 팀 선택 영역에 표시되지 않음 (별도 관전자 섹션)

### 대안: 호스트 배정 모드

스크림 주최자가 팀을 강제 배정해야 하는 경우를 위해,
호스트가 드래그&드롭 또는 "팀 1로 이동" 버튼으로 타인을 배정하는 옵션도 고려 가능.
→ 우선은 **자율 선택 방식**으로 구현 후, 필요 시 확장.

## 백엔드 변경사항

### DB 스키마
- `TeamMode` enum에 `PRESET` 추가
- `RoomParticipant`에 `teamId` nullable 외래키 (이미 존재하면 활용)

### API / 소켓 이벤트
- `select-team` 소켓 이벤트: `{ roomId, teamId }` — 본인을 해당 팀으로 이동
- `leave-team` 소켓 이벤트: 미배정 상태로 복귀
- `team-updated` 브로드캐스트: 팀 변경 시 전체 알림
- 게임 시작 조건: 모든 팀이 정확히 5명

### 서비스 로직
- `startGame` 시 PRESET 모드면 경매/드래프트 스킵 → 바로 역할 선택으로
- 팀은 방 생성 시 자동 생성 (참가자 수 / 5 = 팀 수, 기본 2팀)
- maxParticipants에 맞게 팀 수 결정 (10명=2팀, 15명=3팀 등)

## 프론트엔드 변경사항

- 방 생성 폼에 `PRESET` 팀 모드 옵션 추가
- 로비 페이지에서 `teamMode === "PRESET"`이면 팀 선택 UI 렌더링
- 기존 준비/시작 흐름은 유지 (양 팀 충족 시 호스트가 시작)

## 고려사항

- **팀 수**: maxParticipants 기반으로 자동 계산 (5의 배수)
- **밸런스 경고**: 팀 간 MMR 차이가 클 경우 UI에서 경고 표시 (강제는 아님)
- **중도 이탈**: 게임 시작 전이므로 자유롭게 팀 변경/퇴장 가능
- **봇 미지원**: PRESET 모드에서는 봇 추가 불필요 (실제 팀 구성이므로)

## 우선순위

- 경매/드래프트 UI 개선 완료 후 별도 Phase로 진행 권장
- 핵심 기능(팀 선택 + 역할 선택 연결)만 먼저 구현, 이후 확장
